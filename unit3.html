Китайские слова: прописи по иероглифам

На каждой карточке слово разбито на отдельные иероглифы. Нажми на квадрат с иероглифом — увидишь анимацию порядка черт. Когда анимация закончится, можно сразу писать мышью, стилусом или пальцем поверх клетки.

Карточки сгруппированы по словам со скриншота. Для каждого слова можно отдельно запустить анимацию всех его иероглифов по очереди.

клик по клетке → анимация
после анимации → сразу тренировка
кнопка «Сбросить» очищает текущую пропись
Светлый иероглиф внутри клетки служит опорой для обводки.
Если ошибаться при написании, подсказка по чертам покажется автоматически.
Можно использовать как словарные карточки: сначала нажимай, потом перерисовывай.
